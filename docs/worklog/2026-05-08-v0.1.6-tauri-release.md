# v0.1.6 Tauri 桌面便携包修复发布记录

## Start

- 开始：用户询问是否可以用 `release/risingstones-partyfinder-helper-v0.1.6-desktop-win-x64-portable.zip` 发布国内镜像，并确认 GitHub 是否已经发布。
- 现场核对：本地尚无 `v0.1.6` tag 和 `v0.1.6` zip；GitHub 已有 `v0.1.5` Release，但没有 `v0.1.6` Release。
- 风险：已有 `v0.1.5` 桌面便携资产可能包含误加载 `127.0.0.1:5173` 的坏构建，不适合继续作为最新桌面包。

## Requirement Alignment

- 发布新的补丁版本 `0.1.6`，不覆盖旧 `v0.1.5` 公共资产。
- `0.1.6` 桌面便携包必须开箱即用，启动后加载 Tauri 内嵌前端资源，不依赖 Vite、Node 或本地 Express。
- GitHub 作为公开源码和官方 Release；国内镜像 Release 仍由本机令牌通过 `npm run release:gitee` 上传，令牌和真实镜像地址不写入公开源码或文档。
- 推送维护规则不变：代码和 tag 同步推送到 GitHub 与国内 remote。

## Implementation

- 版本升级到 `0.1.6`：
  - `package.json`
  - `package-lock.json`
  - `src-tauri/Cargo.toml`
  - `src-tauri/Cargo.lock`
  - `src-tauri/tauri.conf.json`
- 更新 README、CHANGELOG、发布说明和项目 Harness 文档入口。
- 继承并发布上一条修复：Tauri 桌面便携构建统一调用 `tauri build --no-bundle`，不再使用裸 `cargo build --release` 生成正式包。
