# Tampermonkey 响应助手说明

## 用途

本地工具不读取官方登录态，也不直接代替账号响应招募。响应助手脚本用于在石之家官方页面内，借用用户已经登录的官方页面环境，提供一个手动响应按钮。

脚本文件：

```text
userscripts/risingstones-response-helper.user.js
```

## 安装

1. 在浏览器安装 Tampermonkey。
2. 新建用户脚本。
3. 将 `userscripts/risingstones-response-helper.user.js` 的内容复制进去。
4. 保存脚本。
5. 打开石之家官方副本招募详情页。

## 使用

在官方招募详情页中：

1. 确认当前页面是你想响应的招募。
2. 点击脚本提供的响应按钮。
3. 输入联系信息。
4. 手动确认提交。

脚本不会自动响应，也不会保存联系信息。

## 错误处理

以下情况会直接展示官方接口返回信息：

- 未登录。
- 未绑定角色。
- 已经响应过。
- 招募已经关闭。
- 官方接口拒绝请求。

## 安全边界

- 脚本只运行在石之家官方域名。
- 脚本不在本地工具页面运行。
- 脚本不读取本地工具存储。
- 脚本不保存账号、Cookie、Token 或联系信息。
- 脚本必须由用户手动确认，不做自动提交。
