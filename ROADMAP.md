# Roadmap

## 已完成第一版

### 可配置更新源

目标：让便携包用户可以从 GitHub 和国内镜像等发布源检查并获取新版本。

当前能力：

- 支持 GitHub Releases。
- 支持本机发布配置注入的国内镜像 Releases。
- 固定维护 GitHub 主仓库和国内镜像两个下载节点。
- 支持加载时 GeoIP 推荐下载节点：中国大陆或检测失败默认国内镜像，未配置国内镜像时回退 GitHub。
- 支持绿/黄/红更新状态灯。
- 默认只“检查并提示”，不静默替换本地文件。
- 下载更新包前展示版本号、发布时间、发布说明和来源地址。
- 不执行未经用户确认的更新包。
- 保留当前版本、最新版本、发布页和附件下载入口。

## 下一优先级

- 安装 Rust/Cargo 后完成 Tauri 桌面客户端 `desktop:build` 原生打包验证。
- 为 Tauri 桌面客户端增加图标、安装器元数据和更新包签名。
- 接入 Tauri updater，实现桌面端一键检查、下载、验证和重启更新。
- 增加 release manifest URL 模式，方便自建镜像源。
- 增加 zip 校验文件展示和校验。
- 便携包保留半自动替换评估，但优先级低于 Tauri 正式客户端。
